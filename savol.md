Quyidagi modelni **professional product specification darajasida** qayta tuzib berdim. Bu siz yozgan g‘oyani **real production onboarding system**ga aylantiradi. Maqsad — **20–30 soniyada maksimal signal olish** va AI uchun **reliable user profile** yaratish.

# High-Signal AI Onboarding System

### (Production-Grade Interest Detection Model)

## Maqsad

Yangi foydalanuvchi haqida **eng muhim signal ma’lumotlarini 20–30 soniya ichida olish** va shu asosida:

* user segment aniqlash
* real maqsad aniqlash
* vaqt resursini baholash
* ishlash uslubini aniqlash
* AI uchun starter profile yaratish

Bu model **cold-start problem**ni hal qiladi.

---

# 1. Segment Classification (Primary Signal)

Bu eng muhim savol.
U **foydalanuvchining asosiy kontekstini** aniqlaydi.

### Savol

> Hozir hayotingizda qaysi yo‘nalish siz uchun eng muhim?

### Variantlar

* 📚 Ta'lim / Talaba
* 💻 Ish / Freelance
* 🚀 Biznes / Startup
* 🧠 Shaxsiy rivojlanish
* 🏃 Sog‘liq / Fitness
* 💰 Moliya / Daromad
* ➕ Boshqa

### Nega muhim

Bu savol orqali AI:

* user segment
* template pack
* mission type

ni aniqlaydi.

Bu savol **profilingning ~70% qismini beradi**.

---

# 2. Goal Extraction (Objective Detection)

Bu savol **konkret maqsadni aniqlaydi**.

### Savol

> Hozir eng ko‘p nimaga erishmoqchisiz?

### Input

Text input yoki suggestion chips.

### Misollar

* IELTS 7.0 olish
* Dasturlash o‘rganish
* biznes boshlash
* vazn tashlash
* yangi ish topish

### AI vazifasi

AI bu inputdan quyidagilarni aniqlaydi:

* goal category
* complexity
* timeline potential

---

# 3. Time Capacity Detection

AI rejalashtirish uchun **eng muhim parametr**.

### Savol

> Kuniga taxminan qancha vaqt ajrata olasiz?

### Variantlar

```
30 minut
1 soat
2 soat
3+ soat
```

### AI uchun ahamiyati

Bu parametr:

* daily workload
* focus sessions
* task density

ni belgilaydi.

---

# 4. Execution Style Detection

Bu savol **user qanday ishlashini aniqlaydi**.

### Savol

> Sizga qaysi usul ko‘proq yordam beradi?

### Variantlar

* 📋 Kunlik tasklar
* 🔁 Odatlar (habits)
* ⏱ Fokus sessionlar
* 🎯 Goal asosidagi rejalash
* ⚡ Aralash model

### AI foydalanishi

Bu parametr:

* planner structure
* habit intensity
* mission generation

ni boshqaradi.

---

# 5. Behavioral Barrier Detection (Optional Advanced Signal)

Bu savol profiling aniqligini sezilarli oshiradi.

### Savol

> Sizni eng ko‘p qiynayotgan narsa nima?

### Variantlar

* procrastination
* vaqt yetishmasligi
* motivatsiya yo‘qligi
* reja tuza olmaslik
* diqqatni jamlay olmaslik

### AI uchun foydalanish

Bu signal:

* coaching tone
* recovery strategy
* plan aggressiveness

ni belgilaydi.

---

# 6. AI Starter Profile Construction

Onboarding tugagach AI quyidagi **structured profile** yaratadi.

### Misol

```json
{
 "segment": "student",
 "goal": "IELTS 7.0",
 "goal_category": "language_learning",
 "daily_capacity": "2h",
 "execution_style": "focus_sessions",
 "main_barrier": "procrastination"
}
```

Bu profil **AI planning system uchun input** bo‘ladi.
